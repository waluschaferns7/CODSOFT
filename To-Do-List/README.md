# 📝 To-Do List GUI Application

A simple and user-friendly To-Do List application built using Python and Tkinter. This project helps users manage daily tasks efficiently with a clean graphical interface.

---

## 🚀 Features

* ➕ Add new tasks
* ✅ Mark tasks as completed
* 🗑️ Delete tasks
* 💾 Persistent storage using JSON
* 🖥️ Interactive GUI using Tkinter

---

## 🛠️ Tech Stack

* **Language:** Python
* **GUI Library:** Tkinter
* **Data Storage:** JSON

---

To-Do-List/

## ▶️ How to Run

1. Clone the repository:

```
git clone https://github.com/waluschaferns7/CODSOFT.git
```

2. Navigate to the project folder:

```
cd To-Do-List
```

3. Run the application:

```
python To-Do-List-GUI.py
```
