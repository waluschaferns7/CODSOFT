# Python Calculator

A clean and modern calculator application built using Python and Tkinter with keyboard support and a custom UI design.

## Features

- Basic arithmetic operations
  - Addition
  - Subtraction
  - Multiplication
  - Division

- Extra functions
  - Square root
  - Square (x²)
  - Percentage
  - Positive/Negative toggle

- User-friendly interface
- Keyboard support
- Backspace button
- Error handling
- Custom color theme

---

## Technologies Used

- Python
- Tkinter

---

## How to Run

1. Install Python
2. Download or clone this repository
3. Run the following command:

```bash
python Calculator.py
```

---

## Keyboard Shortcuts

| Key | Action |
|------|--------|
| 0-9 | Numbers |
| + | Addition |
| - | Subtraction |
| * | Multiplication |
| / | Division |
| Enter | Equals |
| Backspace | Delete last digit |
| . | Decimal |
