# Python Contact Book 📒

A modern **Contact Book Application** built using **Python** and **CustomTkinter** with a clean dark-themed UI and fully functional contact management features.

---

## Features ✨

* Add New Contacts
* Edit Existing Contacts
* Delete Contacts
* Search Contacts Instantly
* Favorite Contacts ⭐
* Input Validation
* Modern Dark UI
* Scrollable Contact List
* Dynamic Contact Display

---

## Technologies Used 🛠️

* Python
* CustomTkinter
* Tkinter Messagebox

## Installation 🚀

### 1. Clone the Repository

```bash
git clone https://github.com/your-username/python-contact-book.git
```

### 2. Navigate to Project Folder

```bash
cd python-contact-book
```

### 3. Install Required Library

```bash
pip install customtkinter
```

### 4. Run the Application

```bash
python main.py
```

---

## Functionalities 📌

### Add Contact

* Save new contacts with:

  * Name
  * Phone Number
  * Email Address

### Edit Contact

* Update existing contact details instantly.

### Delete Contact

* Remove contacts with confirmation popup.

### Search Contacts

* Real-time search filtering.

### Favorite System

* Mark important contacts with a ⭐ star.

### Validations

* Prevents empty fields
* Only allows numeric phone numbers
* Restricts phone number to exactly 10 digits
* Basic email validation included
